# s21Matrix_CPP

Библиотека для работы с матрицами на языке C++, реализующая объектно-ориентированный подход.

## Описание

`s21Matrix_CPP` — это реализация класса `S21Matrix` для выполнения различных операций с матрицами. Библиотека предоставляет удобный интерфейс для математических вычислений, включая арифметические операции, сравнение, транспонирование, вычисление определителя и обратной матрицы.

## Возможности

- **Создание и управление матрицами**:
  - Конструкторы (по умолчанию, параметризованный, копирования, перемещения).
  - Изменение размеров матрицы.
  - Доступ к элементам и их модификация.

- **Арифметические операции**:
  - Сложение и вычитание матриц.
  - Умножение матрицы на число.
  - Умножение двух матриц.

- **Операции сравнения**:
  - Проверка матриц на равенство.

- **Специальные операции**:
  - Транспонирование.
  - Вычисление матрицы алгебраических дополнений.
  - Вычисление определителя.
  - Получение обратной матрицы.

- **Перегрузка операторов**:
  - `+`, `-`, `*`, `+=`, `-=`, `*=`: Арифметические операции.
  - `==`: Сравнение.
  - `=`: Присваивание.
  - `()`: Индексация элементов.

## Сборка и использование

Для сборки проекта используется `Makefile`.

### Основные цели Makefile:

- `make all`: Полная сборка проекта и запуск тестов.
- `make s21_matrix_oop.a`: Сборка статической библиотеки `s21_matrix_oop.a`.
- `make test`: Компиляция и запуск модульных тестов.
- `make clean`: Очистка проекта от временных файлов и объектных файлов.
- `make gcov_report`: Генерация отчета о покрытии кода тестами (требует `lcov`).
- `make cf`: Форматирование кода с помощью `clang-format`.

### Пример использования

```cpp
#include "s21_matrix.h"
#include <iostream>

int main() {
    S21Matrix A(3, 3);
    S21Matrix B(3, 3);

    // Заполнение матриц...
    // A(0, 0) = 1.0;

    S21Matrix C = A + B;
    C.print();

    return 0;
}
```

## Требования

- Компилятор `g++` с поддержкой стандарта C++17.
- `make`.
- Библиотека `check` (для запуска тестов).
